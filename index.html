LOGIC 采配 × オートバトルで、運をねじ伏せて突破

  1. 01 LOGIC — 采配

    戦闘前に、完璧な「論理」を組み立てる

    タンク・魔術師・サポート——役割ごとに個性がある仲間を編成し、 誰をどこに置くかでオートバトルの結果を決める。 ここが LOGIC の入口。操作はこれから始まらない。最初の一手は、采配。

    ROLL で届いたカードや編成の候補を、どう論理に落とし込むか——試行錯誤の余地が、脳汁の源。

  2. 02 LOGIC — オートバトル

    采配した仲間が、自動で動く

    戦闘が始まれば、キャラは自分で挑発し、誘導し、位置を取る—— LOGIC ＝ オートバトル。 あなたは微操に追われず、「論理は正しかったか」を見届ける。

    バラバラに湧く ROLL の敵群も、オートの連携で一か所にまとまる。采配の質が、そのまま戦場の形になる。

  3. 03 ROLL × LOGIC — ねじ伏せ

    運を、一網打尽の快感でねじ伏せる

    まとまった敵群に、範囲魔法が炸裂—— 不確定だった ROLL が、LOGIC によって「想定どおり」に決着した瞬間、 最高の脳汁が走る。

    「運が悪かった」を「采配が勝った」にひっくり返す。だから何度でも、もう一回。
